# Product Guide: GTM Helper

## Initial Concept
Chrome Extension for GTM Helper. Provides useful functions when maintaining and debugging Google Tag Manager Containers.

## Goal
A Chrome Extension to streamline GTM container injection, dataLayer debugging, and CMP consent state management for web analysts and developers.

## Target Audience
- **GTM Users:** Marketing experts testing and debugging GTM containers.
- **Developers:** Web developers implementing and validating CMPs.

## Key Features
- **GTM Injection:** Inject GTM containers on any page for testing.
- **CMP Reset:** Automatically clear cookies and storage for over 60 standard CMPs to reset consent state.
- **GTM Workspace Checkup:** Shortcut to call Analytrix GTM Checkup directly from a GTM workspace.
- **DataLayer Tools:** Push events and execute custom JS on selected pages.
- **Third-Party Integration:** Shortcuts to Analytrix GTM Checkup documentation and CMP list.

## Platform
- Primary: **Google Chrome** (Chrome Web Store).

## Visual Identity
- **Engineering Aesthetic:** System fonts, primary color Orange (#C44E00), interaction color Green (#005141).
